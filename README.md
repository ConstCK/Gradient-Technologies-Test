# Gradient-Technolgies — сервис сокращения ссылок

API для создания коротких ссылок, редиректа и статистики переходов (FastAPI, PostgreSQL, SQLAlchemy 2 async).

Базовый путь API: `/api/v1`.

1. **GET** `/api/v1/health` — проверка доступности сервиса (`{"status": "ok"}`).
2. **POST** `/api/v1/shorten` — сократить ссылку. Тело: `{"url": "https://..."}`. Ответ: `short_id` (с префиксом `gradient-technologies/`), `original_url`.
3. **GET** `/api/v1/stats/{short_id}` — статистика по короткой ссылке: `short_id`, `clicks`. Идентификатор можно передать с префиксом или без.
4. **GET** `/api/v1/{short_id}` — редирект (302) на оригинальный URL. Учитывается как переход (клик).

**Под капотом:** для `short_id` в п. 3 и 4 префикс `gradient-technologies/` при необходимости снимается, поиск в БД по короткому ключу. **Shorten:** сначала проверка по исходному URL — если ссылка уже есть, возвращается существующая; иначе генерируется уникальный короткий id (секретный токен), создаётся запись в БД. Один URL всегда даёт один и тот же short_id. **Stats:** выборка по short_id из БД, возврат счётчика кликов. **Редирект:** поиск по short_id, инкремент счётчика кликов, ответ 302 на сохранённый original_url; при отсутствии ссылки — 404.

Полная документация после запуска: http://localhost:8000/docs.

## Запуск

**Через .env (локально):**

1. Скопировать `.env.example` в `.env`, указать `DATABASE_URL` (PostgreSQL с asyncpg).
2. Создать БД, выполнить миграции: `alembic upgrade head`.
3. Запуск: `python main.py`.

**Через Docker:**

```bash
docker compose up --build
```

Поднимаются postgres, приложение (миграции + uvicorn) и сервис **tests** — автоматически запускаются pytest; после прогона контейнер tests завершается. Приложение: http://localhost:8000, документация: http://localhost:8000/docs. Postgres — порт 5432.
